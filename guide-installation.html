Flashcards App — Guide d'installation

Créez votre propre application de flashcards anglais-français
en 3 étapes simples • Gratuit • Sans abonnement

🃏
Ce que vous avez reçu
Contenu du package
  • flashcards-vierge.html — Le fichier principal de l'application (à personnaliser)
  • guide-installation.html — Ce guide (vous y êtes !)
💡 L'application fonctionne déjà en local sur votre ordinateur, sans installation. Ouvrez simplement flashcards-vierge.html dans votre navigateur pour tester !
1
Clé API Pixabay Gratuit
Pour les suggestions d'images automatiques
  1. Allez sur pixabay.com → créez un compte gratuit
  2. Allez sur pixabay.com/api/docs
  3. Votre clé API apparaît en haut de la page — copiez-la
  4. Ouvrez flashcards-vierge.html dans un éditeur de texte (TextEdit sur Mac, Notepad sur Windows)
  5. Cherchez ce texte et remplacez-le par votre clé :
const PIXABAY_KEY = 'VOTRE_CLE_PIXABAY'; → remplacer par : const PIXABAY_KEY = '12345678-abcdef1234567890abcdef12';
2
Clé API Mistral AI Gratuit
Pour la génération de listes de vocabulaire par IA
  1. Allez sur console.mistral.ai
  2. Créez un compte gratuit (email suffit)
  3. Menu « API Keys » → « Create new key »
  4. Copiez la clé générée
  5. Dans flashcards-vierge.html, remplacez :
const MISTRAL_KEY = 'VOTRE_CLE_MISTRAL'; → remplacer par votre clé Mistral
3
Base de données Firebase Gratuit
Pour synchroniser vos données sur tous vos appareils
  1. Allez sur firebase.google.com → connectez-vous avec Google
  2. Cliquez « Créer un projet » → choisissez un nom → désactivez Analytics → « Créer »
  3. Dans le menu gauche → « Realtime Database » → « Créer une base de données »
  4. Choisissez « Mode test » → région europe-west3 → « Activer »
  5. Cliquez l'icône ⚙️ → « Paramètres du projet » → descendez jusqu'à « Vos applications »
  6. Cliquez l'icône </> (Web) → nommez l'app → « Enregistrer »
  7. Copiez le bloc firebaseConfig qui apparaît
  8. Dans flashcards-vierge.html, remplacez le bloc firebaseConfig par le vôtre :
const firebaseConfig = { apiKey: "VOTRE_API_KEY_FIREBASE", authDomain: "VOTRE_PROJET.firebaseapp.com", databaseURL: "https://VOTRE_PROJET-default-rtdb...", projectId: "VOTRE_PROJET", ... };
⚠️ N'oubliez pas d'aller dans Realtime Database → Règles et de mettre .read: true, .write: true puis cliquez Publier.
4
Déploiement sur Cloudflare Pages Gratuit
Pour mettre votre application en ligne
  1. Allez sur pages.cloudflare.com
  2. Créez un compte gratuit
  3. Cliquez « Create a project » → « Upload assets »
  4. Nommez votre projet (ex : flashcards-dupont)
  5. Glissez-déposez votre fichier flashcards-vierge.html — renommé en index.html
  6. Cliquez « Deploy site »
  7. Votre URL sera : flashcards-dupont.pages.dev ✅
💡 Pour mettre à jour votre application plus tard : revenez sur Cloudflare → votre projet → « Create deployment » → re-déposez votre index.html modifié.
🏠
Utilisation au quotidien
Comment utiliser votre application
  • Interface enseignant : ouvrez votre URL → l'interface enseignant s'ouvre directement → créez vos classes et paquets de cartes
  • Partager avec les élèves : dans un paquet → bouton 🔗 Partager → copiez le lien → envoyez-le aux élèves
  • Générer des cartes avec l'IA : dans un paquet → bouton ✨ Générer avec l'IA → entrez un thème et le nombre de cartes souhaité
  • Importer une liste : dans un paquet → 📥 Importer une liste → collez votre liste depuis Excel ou Word
  • Sauvegarde : bouton 🔁 Transfert → 📤 Exporter → téléchargez un fichier .json de sauvegarde

✅ Récapitulatif — Tout ce que vous devez faire

  • Créer un compte Pixabay → récupérer la clé API
  • Créer un compte Mistral AI → récupérer la clé API
  • Créer un projet Firebase → récupérer le firebaseConfig → activer les règles
  • Modifier les 3 éléments dans flashcards-vierge.html
  • Renommer le fichier en index.html
  • Créer un compte Cloudflare Pages → déployer le fichier
  • Votre app est en ligne ! 🎉

⏱️ Temps estimé : 20-30 minutes pour la première installation
